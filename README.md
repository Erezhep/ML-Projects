# My Machine Learning Journey 🚀

Бұл репозиторийде менің барлық ML және AI жобаларым жинақталған.

## Жобалар тізімі:

### 1. MNIST Digit Recognizer (CNN) 🔢
* **Сипаттама:** Қолмен жазылған сандарды 99% дәлдікпен танитын модель.
* **Технологиялар:** Python, TensorFlow/Keras, Flask, OpenCV.
* **Қолданылуы:** [Сілтеме осы папкаға](./01-MNIST-Digit-Classification/)

### 2. Medical Cost Prediction (Linear/Ridge Regression, Polynomial features) 🏠
* **Сипаттама:** Пациенттің параметрлеріне (жасы, BMI, шылым шегуі) қарап, оның медициналық сақтандыру шығындарын болжайтын модель.
* **Технологиялар:** Scikit-learn, Pandas, Joblib, Matplotlib, Seaborn.
* **Қолданылуы:** [Сілтеме осы папкаға](./02-Medical-Cost-Prediction/)

### 3. Heart Disease Classification (Logistic Regression) 🩺
* **Сипаттама:** Пациенттің медициналық көрсеткіштеріне (жасы, қан қысымы, холестерин деңгейі) сүйене отырып, жүрек ауруының бар-жоғын және оның ықтималдығын болжайтын классификациялық модель.
* **Технологиялар:** Scikit-learn, Pandas, Joblib, Matplotlib, Seaborn.
* **Қолданылуы:** [Сілтеме осы папкаға](./03-Heart-Disease-Prediction/)

### 4. KNNeighbors Classifier (KNN) ❄️
* **Сипаттама:** Гүлдің 4 түрлі физикалық өлшеміне (сепал/петал ұзындығы мен ені) сүйене отырып, оның қай түрге (Setosa, Versicolor, Virginica) жататынын анықтайтын классификациялық модель.
* **Технологиялар:** Scikit-learn, Pandas, Joblib, Matplotlib, Seaborn.
* **Қолданылуы:** [Сілтеме осы папкаға](./04-KNN-Classifier-Iris-Data/)

### 5. Decision Tree Classifier (Шешім ағаштары) 🌳
* **Сипаттама:** Титаник жолаушыларының жеке деректеріне (жынысы, жасы, билет класы мен құны) сүйене отырып, логикалық шешімдер тізбегі арқылы олардың тірі қалу-қалмауын болжайтын классификациялық модель. Модель деректерді иерархиялық тармақтарға бөлу арқылы шешім қабылдайды.
* **Технологиялар:** Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn, Joblib.
* **Қолданылуы:** [Сілтеме осы папкаға](./05-Decision-Tree-Titanic/)


### 6. Naive Bayes Classifier (Наивті Байес) 📧
* **Сипаттама:** Байес теоремасына негізделген, SMS хаттардың мәтінін талдау арқылы олардың "спам" немесе "таза хат" (ham) екенін анықтайтын классификациялық модель. Алгоритм сөздердің кездесу жиілігіне сүйеніп, әрбір белгіні бір-біріне тәуелсіз деп есептейтін "аңқау" (naive) принципі арқылы ықтималдықтарды есептейді. Мәтіндік деректерді (NLP) өңдеуде өте жоғары жылдамдық пен тиімділік көрсетеді.
* **Технологиялар:** Scikit-learn (MultinomialNB, CountVectorizer), Pandas, NumPy, Matplotlib, Seaborn.
* **Қолданылуы:** [Сілтеме осы папкаға](./06-Naive-Bayes-SMS-Spam/)
